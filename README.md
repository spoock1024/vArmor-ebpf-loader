# vArmor-ebpf 加载器
加载和测试 [vArmor-ebpf](https://github.com/bytedance/vArmor-ebpf) 项目中的 eBPF 程序。当前已经实现测试的函数如下：
- [x] varmor_bprm_check_security
- [x] varmor_capable
- [x] varmor_file_open
- [x] varmor_mount
- [x] varmor_path_link
- [x] varmor_path_rename
- [x] varmor_path_symlink
- [x] varmor_socket_connect
- [x] varmor_umount
